  # 1.1 디자인 패턴

  ### 1.1.1 싱글톤 패턴
  싱글톤 패턴(Singleton pattern)은 하나의 클래스에 오직 하나의 인스턴스만 가지는 패턴입니다. 보통 데이터베이스 연결 모듈에 많이 사용합니다. 
  하나의 인스턴스를 만들어 놓고 해당 인스턴스를 다른 모듈들이 공유하며 사용하기 때문에 인스턴스를 사용할 때 드는 비용이 줄어드는 장점이 있습니다.
  하지만 의존성이 높아진다는 단점이 있습니다.
